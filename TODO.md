# Portfolio Tasks

## 🎯 When Creating Real Projects

- [ ] Update project links in `src/utils/data.js`
- [ ] Add real project images to `public/images/projects/`
- [ ] Replace placeholder URLs (`yourportfolio.com`) with your domain
- [ ] Update Twitter handle in `src/utils/seo.js`
- [ ] Update `VITE_API_URL` for production

## 🚀 Before Deployment

- [ ] Set up Gmail app password in `server/.env`
- [ ] Deploy frontend to Vercel/Netlify
- [ ] Deploy backend to Railway/Render
- [ ] Update `CORS_ORIGIN` in production `.env`
- [ ] Test contact form on live site
- [ ] Set custom domain DNS records
